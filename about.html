About this app

This project helps compare a personal BVB portfolio against the BET index distribution, and suggests how to invest available cash to move the portfolio closer to the index (without selling positions).

Live app: https://dragosgeornoiu.github.io/bvb-index-distribution/

How it works

  • BET weights are fetched from a daily-scraped CSV committed by a GitHub Action.
  • Portfolio is provided as a CSV in symbol,value format (values in your base currency).
  • CASH_VALUE is treated as available cash and excluded from portfolio weights.
  • The app compares portfolio weights vs normalized index weights and shows differences.
  • If you enable “New Investment”, it proposes buy-only allocations that keep the total ≤ available cash.
  • Allocations respect a configurable minimum per stock and are rounded down to lots of 50.
  • A second table shows portfolio after applying suggestions so you can validate improvement.

Input formats

Portfolio CSV (upload in the Tracker tab):

symbol,value
SNP,126057.38
TLV,121373.78
...
CASH_VALUE,12023.80

Index CSV (auto-fetched):

symbol,weight
TLV,19.30
SNP,19.03
...
