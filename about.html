About this app

This project helps compare a personal BVB portfolio against the BET index distribution and suggests how to invest available cash to move the portfolio closer to the index — using buy-only allocations (no selling).

Live app: https://dragosgeornoiu.github.io/bvb-index-distribution/

How it works

  • BET weights are fetched from a daily-scraped CSV committed by a GitHub Action.
  • Your portfolio can be imported either via CSV upload or via TradeVille copy/paste.
  • CASH_VALUE represents available cash and is excluded from portfolio weights.
  • The app compares portfolio weights vs normalized index weights.
  • If New Investment is enabled, the app proposes buy-only allocations.
  • Allocations respect a configurable minimum per stock and are rounded down to lots of 50.
  • A second table shows portfolio after applying suggestions for validation.

Portfolio input formats

1) Portfolio CSV upload

Upload a CSV file with the following format:

symbol,value
SNP,126057.38
TLV,121373.78
SNG,75962.08
...
CASH_VALUE,12023.80
  • symbol — BVB ticker
  • value — current position value (not percentage)
  • CASH_VALUE — available cash (optional)

2) TradeVille copy / paste import

You can also paste your portfolio table directly from TradeVille. The parser is intentionally tolerant and ignores extra text.

Supported pasted content includes:

  • Table headers
  • Extra rows before or after the table
  • “Total active”, “Total Portofoliu”, and similar summary rows

Example (direct copy from TradeVille):

simbol      Nume                    sold   Pret piata   evaluare   Pondere
SNP         OMV Petrom              128,368 0.9760       125,287.17 20.57%
TLV         Banca Transilvania      4,027   30.22        121,695.94 19.98%
SNG         Romgaz                  7,783   9.78         76,117.74  12.50%
...
RON         RON                     12,023.80

The app automatically:

  • extracts symbols and evaluated values
  • detects the RON row and maps it to CASH_VALUE
  • ignores any rows it cannot safely parse
